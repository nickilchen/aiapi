## 🚀 使用步骤

### 1. 重启服务
```bash
npm start
```

### 2. 访问告警页面
打开 http://localhost:3000，点击侧边栏"告警通知"

### 3. 配置企业微信通知

#### 获取 Webhook URL
1. 在企业微信群中，点击右上角 "..." → "群机器人"
2. 点击"添加机器人"
3. 设置机器人名称和头像
4. 复制 Webhook 地址

#### 配置告警系统
1. 开启"企业微信通知"开关
2. 粘贴 Webhook URL
3. （可选）填写需要 @的成员 userid 或手机号
4. 点击"保存配置"

### 4. 测试告警
点击"测试告警"按钮，检查企业微信群是否收到消息
