## 📝 注意事项

### 1. 多语言支持
虽然当前主要使用中文，但系统仍保留了多语言支持能力：
- 所有文本都通过 `data-i18n` 属性绑定
- 可以轻松添加其他语言翻译
- 用户可以通过语言切换器切换语言

### 2. Placeholder 处理
系统使用 `data-i18n-placeholder` 属性来翻译 placeholder：
```html
<input data-i18n-placeholder="alerts.notifications.wecomWebhookUrlPlaceholder">
```

i18n.js 会自动处理这个属性，将翻译应用到 placeholder。

### 3. 动态内容
JavaScript 中的动态内容使用 `t()` 函数：
```javascript
showToast(t('common.success'), '告警配置已保存', 'success');
```

### 4. 严重程度文本
严重程度文本在 JavaScript 中硬编码为中文：
```javascript
function getSeverityText(severity) {
    const texts = {
        info: '信息',
        warning: '警告',
        error: '错误'
    };
    return texts[severity] || severity;
}
```
